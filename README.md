## About The Project

Project Duration: Jan 10, 2024 - July 06, 2024

INFOU 는 Inha Univ. 내부 강의평과 공지사항 데이터를 한눈에 확인할 수 있는 어플리케이션 서비스 입니다. 

&nbsp;

### Key Feature
- **강의평 데이터:** 수강신청 시 필수 요구되는 강의 만족도 평가를 가져옵니다.

- **공지사항 추천:** 사용자와 비슷한 학과,학년,관심도 정보를 기반으로 필요한 공지사항을 추천해줍니다.
